# Set Up

 ## Server

 - Open `cmd`
 - Go to repository `~/server` folder
 - Run `npm install`
 - Run `npm start`
 
 ### Server host

 - http://localhost:3000/

## App

 - Open `cmd`
 - Go to repository root
 - Run `npm install`
 - Run `npm start`

 ### App host

 - http://localhost:4200/ 

 ## Database

 - Open PhpMyAdmin
 - Create a new database, name is `project-manager`
 - Import test data. (`~/server/data/`)
